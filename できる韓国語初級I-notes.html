② 助词总览

韩语助词接在名词后，标注名词在句中的作用，类似中文的"是……的"、"把"、"到"等。

核心助词

助词中文对应用法规则例
은/는 主题标记「は」 尾音有辅音→은，无辅音→는 사람은 / 회사는
이/가 主语标记「が」 有辅音→이，无辅音→가 학생이 / 교사가
을/를 宾语标记「を」 有辅音→을，无辅音→를 책을 / 커피를
도 也、还 无辅音变化 저도 학생이에요
만 只，仅 无辅音变化 가족만 입어요

位置与时间助词

助词中文对应说明例
에 在……（位置）/在……（时间） 表示静态位置或时间点 방에 있어요 / 6시에
에서 在……（动作发生地）/ 从…… 表示动作发生地点或出发地 학교에서 공부해요
부터 从……（时间起点） 时间起点用부터，场所起点用에서 9시부터
까지 到……（终点） 时间和场所都可用 6시까지 / 서울까지
에서…까지 从……到……（场所） 집에서 학교까지
부터…까지 从……到……（时间） 9시부터 6시까지

其他常用助词

助词中文对应用法规则例
하고 / 와/과 和……（与……一起） 하고口语，와/과书面；有辅音→과，无辅音→와 친구하고 가요
(으)로 用……（手段）/ 往……（方向） 有辅音→으로（ㄹ→로），无辅音→로 버스로 가요 / 기차로
보다 比……（比较） 无辅音变化 서울보다 따뜻해요
에게 / 한테 给……（对人） 에게书面，한테口语 친구에게 주다
에게서 / 한테서 从……（人）那里得到 친구한테서 받았어요
께 / 께서 给……（敬语）/ 主语（敬语） 께=에게敬，께서=이/가敬 선생님께서 오세요
의 ……的（所属） 实际发音常读成 [에] 나의 가방
(이)라고 합니다 叫做……（自我介绍名字） 有辅音→이라고，无辅音→라고 다나카라고 합니다

④ 语法句型（第1~20课）

每课的核心句型，配有简明中文说明和例句。

1
입니다 / 입니까?
我是…… / 是……吗？（正式体）
名词 + 입니다
「是……」——正式、公开场合用（发音：[임니다]）
저는 회사원입니다. — 我是上班族。
名词 + 입니까?
「是……吗？」——正式疑问（发音：[임니까]）
학생입니까? — 是学生吗？
은/는（主题助词）
类似中文"说到……"，标出句子主题；有尾音→은，无尾音→는
사람은（说到人）/ 회사는（说到公司）

● 이라고 합니다：我叫做……（自我介绍名字时）

例：다나카라고 합니다. — 我叫田中。

2
이/가 아닙니다
不是……（正式体否定）
名词 + 이/가 아닙니다
「不是……」——有尾音用이，无尾音用가
한국 사람이 아닙니다. — 不是韩国人。
배우가 아닙니다. — 不是演员。
이/가（主语助词）
标记主语；有尾音→이，无尾音→가
일본인이（日本人是）/ 회사가（公司是）
3
예요 / 이에요
是……（日常口语体）
名词（元音结尾）+ 예요
「是……」——日常对话用，比입니다更随意
교사예요. — 是老师。/ 서울이에요. — 是首尔。
名词（辅音结尾）+ 이에요
同上，辅音结尾用이에요
학생이에요? — 是学生吗？
이/가 아니에요
「不是……」——日常否定
교사가 아니에요. — 不是老师。
指示代词单独+은/는+이/가
이 这个（近）이것/이거이것은/이건이것이/이게
그 那个（近对方）그것/그거그것은/그건그것이/그게
저 那个（远处）저것/저거저것은/저건저것이/저게
4
있습니다 / 없습니다
有/在 / 没有/不在
있습니다 / 있어요
「有……」/ 「在……」
친구가 있어요. — 有朋友。/ 식당이 있습니다. — 有餐厅。
없습니다 / 없어요
「没有……」/ 「不在……」
시간이 없어요. — 没时间。/ 친구가 없습니다. — 没有朋友。
5
에 있어요（位置表达）
在哪里？
场所 + 에 있어요
「在……里/上/前……」，位置助词用「에」
책상 위에 있어요. — 在书桌上面。/ 가방은 어디에 있어요? — 包在哪里？
韩语中文韩语中文
위上面밑/아래下面
옆旁边앞前面
뒤后面안里面
밖外面오른쪽右边
왼쪽左边맞은편对面
근처附近
6
ㄴ/습니다 · -고
动词·形容词正式体 / 连接「和、然后」
词干 + ㄴ니다（无尾音）/ 습니다（有尾音）
动词·形容词的正式礼貌体（类似日语「ます」），疑问→ ㄴ니까? / 습니까?
가다 → 갑니다 / 갑니까?（去/去吗？）
먹다 → 먹습니다 / 먹습니까?（吃/吃吗？）
⚠ ㄹ结尾词干：ㄹ脱落 → 살다 → 삽니다
词干 + 고
连接两个动作「……，然后……」/ 「……和……」
청소를 하고 빨래합니다. — 打扫然后洗衣服。
7
否定句
不……
안 + 동사/형용사
短否定「不……」（안放在动词前）
안 갑니다. — 不去。/ 안 먹어요. — 不吃。
词干 + 지 않습니다 / 지 않아요
长否定，意思相同，语气略正式
가지 않습니다. — 不去。/ 멀지 않아요. — 不远。

⚠ 「하다」动词的否定：안放在名词前，不放在하다前

✗ 안 공부합니다 → ✓ 공부를 안 합니다（不学习）

8
-네요
啊，真是……呢！（感叹·同感）
词干 + 네요
表达感叹、共鸣，类似中文「……呢！」「好……啊！」；ㄹ词干脱落
싸네요! — 好便宜啊！
맛있네요! — 真好吃啊！
멀다 → 머네요（ㄹ脱落）
9
-아/어요（日常体）
比ㄴ/습니다更口语的礼貌体
词干母音是 ㅏ/ㅗ → + 아요
阳性母音（ㅏ/ㅗ）词干，加「아요」
살다 → 살아요（住）/ 좋다 → 좋아요（好）
词干其他母音 → + 어요
阴性母音词干，加「어요」
먹다 → 먹어요（吃）/ 맛있다 → 맛있어요
하다 → 해요
所有하다动词的口语体
공부하다 → 공부해요（学习）/ 유명하다 → 유명해요（有名）

● 词干末尾是元音时，与아/어合并缩略：

가다→가요 / 오다→와요 / 배우다→배워요 / 기다리다→기다려요

11
-았/었 (过去式)
……了 / ……过
词干（ㅏ/ㅗ）+ 았습니다 / 았어요
阳性母音词干的过去形
살다 → 살았어요（住过）/ 좋았어요（很好）
词干（其他）+ 었습니다 / 었어요
阴性母音词干的过去形
먹다 → 먹었어요（吃了）
하다 → 했습니다 / 했어요
하다动词的过去形
공부했어요. — 学习了。/ 갔어요.（가다→가+았→갔）— 去了。
名词 + 이었어요 / 였어요
名词过去形「曾是……」；有尾音→이었어요，无尾音→였어요
학생이었어요. — 曾是学生。
12
敬语 -(으)시-
对长辈或地位高的人使用
词干 + (으)세요 / (으)십니다
敬语体；有尾音词干加으시，无尾音加시；ㄹ词干ㄹ脱落
가다 → 가세요?（您去吗？）
읽다 → 읽으세요?（您看吗？）
알다 → 아세요?（ㄹ脱落，您知道吗？）
普通敬语中文
먹다 / 마시다드시다吃/喝（敬）
있다계시다在（敬）
자다주무시다睡觉（敬）
죽다돌아가시다去世（敬）
-(으)세요 / -(으)십시오（命令/请求）
「请……」；(으)십시오 用于更正式公共场合
앉으세요. — 请坐。/ 기다려 주십시오. — 请等候。
13
敬语过去形 -(으)셨어요
您（做了）……吗？
词干 + (으)셨습니다 / (으)셨어요
敬语过去形；规则同敬语现在形，只是시→셨
가셨어요? — 您去了吗？
드셨어요? — 您吃了吗？（드시다的过去）
14
-지요? · -지만
……对吧？ / ……但是……
词干 + 지요? （口语缩略：죠?）
向对方确认「……对吧？」「……是吧？」
내일 가지요? — 明天去吧？/ 회사원이지요? — 是上班族对吧？
词干 + 지만
逆接连词「虽然……但是……」
맛있지만 비싸요. — 好吃但很贵。/ 주말이지만 바빠요. — 虽然是周末但很忙。
15
-고 싶어요
想要……（愿望）
词干 + 고 싶어요 / 고 싶습니다
「我想……」（说话人自己的愿望）
온천에 가고 싶어요. — 我想去温泉。
词干 + 고 싶지 않아요
「我不想……」
여기에 있고 싶지 않아요. — 我不想待在这里。
词干 + 고 싶어 해요
「他/她想……」（第三人称愿望用此形式）
친구가 가고 싶어 해요. — 朋友想去。
16
-아/어서（理由）· 으不规则
因为……所以……
词干 + 아서 / 어서（하다→해서）
表示理由原因「因为……，所以……」；⚠ 아/어서前面不能用过去形
바람이 불어서 춥습니다. — 因为刮风所以冷。
너무 맛있어서 또 먹었어요. — 因为太好吃，又吃了一次。（✓）
✗ 맛있었어서 — 错误，어서前不用었
으不规则：词干末音ㅡ + 아/어 → ㅡ脱落
词干末是「ㅡ」的动词，加아/어时ㅡ脱落，根据前一母音决定아/어
바쁘다 → 바빠요 / 바빠서（忙）
예쁘다 → 예뻐요 / 예뻐서（漂亮）
쓰다 → 써요 / 써서（用/写）
아프다 → 아파요（疼）/ 크다 → 커요（大）
17
许可·条件·假设
可以吗？/ 如果……就……
词干 + 아/어도 돼요?
「……可以吗？」（征求许可）
같이 가도 돼요? — 可以一起去吗？
回答：네, 아/어도 돼요 / 안 돼요 / (으)면 안 돼요
肯定：可以。/ 否定：不行。/ 条件否定：……的话不行。
여기서 사진을 찍으면 안 돼요. — 这里不能拍照。
词干 + (으)면 돼요
「……就行了」（条件成立即可）
9시까지 오면 돼요. — 9点前来就行了。
词干 + (으)면
「如果……」「……的话」（条件/假设）
돈이 있으면 여행하고 싶어요. — 如果有钱就想旅行。
18
建议·目的
我们……吧 / 为了……去……
词干 + (으)ㄹ까요?
「我们……吧？」（提议，询问对方意见）；ㄹ词干脱落
커피라도 마실까요? — 喝点咖啡怎么样？/ 뭘 만들까요? — 做什么好呢？
词干 + (으)ㅂ시다 / 아/어요
「我们……吧！」（劝诱；不对长辈使用此形式）
빨리 갑시다. — 快走吧！/ 가요. — 走吧。
词干 + (으)러 + 移动动词
「为了……而去/来」（目的；后接가다/오다等移动动词）
배우러 갑니다. — 去学习。/ 서류를 받으러 왔습니다. — 来取文件。
19
请求表达
麻烦您……/ 请您……
词干 + 아/어 주시겠어요?
「能不能请您……？」（有礼貌的委托，常加좀）
사진 좀 찍어 주시겠어요? — 能帮我们拍张照吗？
词干 + 아/어 주세요 / 아/어 주십시오
「请……」（请求/拜托）；주십시오更正式
창문을 열어 주세요. — 请开一下窗户。
ㄴ/습니다만 · 죄송합니다만 · 실례합니다만
「……不过……」，请求前的客套前缀，相当于「对不起，打扰一下……」
죄송합니다만, 좀 기다려 주시겠어요? — 抱歉，能稍等一下吗？
20
能力·可能
会……吗？/ 不会……
词干 + (으)ㄹ 수 있어요
「会……」「能……」（表示能力/可能）
자전거를 탈 수 있어요? — 会骑自行车吗？
못 + 动词 / 词干 + 지 못해요
「不会……」「不能……」（不可能/无法做到）
자전거를 못 타요. — 不会骑自行车。

⑦ 发音规则

实际发音与书写形式不同时，需遵循以下规则。

规则名说明例
연음화 连音化 收音（尾音）后接元音时，收音移到下一个音节 사람이→[사라미]　읽어요→[일거요]
비음화 鼻音化 ㅂ/ㄷ/ㄱ后接鼻音(ㄴ/ㅁ)时→ ㅁ/ㄴ/ㅇ 입니다→[임니다]　학년→[항년]
농음화 紧音化 收音ㅂ/ㄷ/ㄱ后接ㄱ/ㄷ/ㅂ/ㅈ/ㅅ→后者变紧音 학교→[학꾜]　약속→[약쏙]
격음화 激音化 ㅎ与相邻辅音ㄱ/ㄷ/ㅈ/ㅂ合流→ ㅋ/ㅌ/ㅊ/ㅍ 좋다→[조타]　어떻게→[어떠케]
유음화 流音化 ㄴ+ㄹ或ㄹ+ㄴ → 都变[ㄹㄹ] 일 년→[일련]　한류→[할류]
구개음화 腭音化 ㄷ/ㅌ后接이→变[지/치] 같이→[가치]　굳이→[구지]
ㅎ약화 H弱化 有声音之间的ㅎ发音变弱或消失 전화→[저놔]　결혼→[겨론]
